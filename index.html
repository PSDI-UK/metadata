PSDI metadata

Overview of PSDI metadata

Summary

The PSDI metadata for the Physical Sciences Data Infrastructure (PSDI) project.

The metadata is made available in JSON-LD format:

  • vocabulary.jsonld corresponds to the Cross-Domain Interoperability Framework (CDIF) CORE CONTROLLED VOCABULARY profile
  • resource-catalogue.jsonld corresponds to the Cross-Domain Interoperability Framework (CDIF) CORE DISCOVERY profile
